## 📌 项目简介

**微信易多开** 是一款轻量级、免安装、绿色安全的 Windows 微信多开工具。  
无需修改注册表、无需第三方注入，只需拖拽微信快捷方式或 exe 文件，即可一键实现 **单开 / 双开 / 三开 / N开**！

✅ 自动识别微信路径  
✅ 支持手动设置 / 自动检测  
✅ 界面简洁，操作直观  
✅ 无广告、无捆绑、无后门

## ✅ 功能特性

- **一键多开**：支持 1~N 个微信实例同时运行（默认支持 1/2/3/N×）
- **智能识别**：自动检测 `Weixin.exe` / `WeChat.exe` 路径
- **灵活配置**：支持拖拽快捷方式、手动设置路径、自动检测三种方式
- **安全稳定**：不修改系统文件，不注入进程，纯启动器机制
- **轻量快速**：启动秒开，资源占用极低

## 🛠️ 使用方法

### 方法一：自动检测（推荐）

1. 打开软件，点击 **「自动检测」**
2. 程序会自动扫描常见路径（如 `C:\Program Files (x86)\Tencent\WeChat\`）
3. 检测成功后，点击对应按钮（1单开 / 2双开 / 3三开 / N×5）即可

### 方法二：手动设置

1. 点击 **「手动设置」**
2. 浏览并选择你的 `Weixin.exe` 或 `WeChat.exe` 文件路径
3. 设置完成后，点击对应多开按钮

### 方法三：拖拽快捷方式

1. 将桌面微信快捷方式直接拖入绿色区域
2. 程序自动识别路径并显示在输入框
3. 点击「多开」按钮即可

## 📦 安装与运行

> ⚠️ 本项目为绿色软件，**无需安装**，解压即用！

1. 下载最新版 `微信易多开_vX.X.zip`

## 📝 常见问题 FAQ

### Q1：为什么检测不到微信路径？

> A：请确保微信已安装，且路径中包含 `Weixin.exe` 或 `WeChat.exe`。也可手动设置路径。

### Q2：多开后微信会卡顿吗？

> A：不会。每个微信实例都是独立进程，互不干扰，性能取决于你的电脑配置。

### Q3：支持 Mac / Linux 吗？

> A：目前仅支持 Windows。Mac/Linux 版本正在开发中（欢迎贡献代码）。

### Q4：是否会被微信封号？

> A：本工具仅为启动器，**不修改微信客户端**，官方未禁止多开行为，风险极低。（但仍建议遵守微信用户协议）
